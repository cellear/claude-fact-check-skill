# Fact-Check Examples

This directory contains example fact-checks demonstrating the methodology in action.

## Examples

### [Example 01: Coffee and Cancer Risk](example-01-coffee-cancer.md)

**Claim Type**: Health/Science misinformation  
**Complexity**: Medium  
**Key Learning**: Demonstrates how legitimate research gets distorted through cherry-picking and sensationalism

**Topics Covered**:
- Source independence analysis (tracing viral claims back to original research)
- Cherry-picking detection (subgroup finding misrepresented as overall finding)
- Vague authority appeals ("researchers at major university")
- Headline-text mismatch
- Science misrepresentation

---

## How to Use These Examples

1. **Learning the Methodology**: Read through examples to see how each step of the fact-checking process works in practice

2. **Training**: Use these as templates when conducting your own fact-checks

3. **Teaching**: Share with others to demonstrate critical thinking and media literacy

4. **Comparison**: Try fact-checking the same claims yourself before reading the examples

## Contributing Examples

Have a great fact-check example? Contributions welcome! See [CONTRIBUTING.md](../CONTRIBUTING.md) for guidelines.

Good examples should:
- Cover diverse claim types (health, politics, science, finance, etc.)
- Demonstrate different aspects of the methodology
- Show both clear-cut and nuanced cases
- Include real or realistic scenarios
- Protect privacy (anonymize sources when needed)
